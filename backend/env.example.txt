# ----------------------------------------
# Google OAuth Configuration
# ----------------------------------------
# We use ONLY the Web Client ID for all platforms (Android, iOS, Web)
# This uses Expo's browser-based OAuth flow which works everywhere
# without needing SHA-1 fingerprints or separate client IDs
#
# Get your Web Client ID from Google Cloud Console:
# 1. Go to https://console.cloud.google.com
# 2. Create a new project or select existing
# 3. Go to APIs & Services > Credentials
# 4. Create OAuth 2.0 Client ID (Application type: Web application)
# 5. Add authorized redirect URI: https://auth.expo.io/@your-expo-username/your-app-slug
#
SPLITBILL_GOOGLE_CLIENT_ID=your-google-web-client-id.apps.googleusercontent.com
SPLITBILL_GOOGLE_SECRET_KEY=your-google-client-secret
